# 슬기로운 코워크 생활 (Smart Cowork Life)

> 한국 직장인·취준생·1인사업자를 위한 AI 업무 자동화 플러그인

---

## 프로젝트 개요

**슬기로운 코워크 생활**은 Claude Code 및 Cline 환경에서 동작하는 듀얼 포맷 AI 스킬 플러그인입니다. 한국 직장 문화에 특화된 12가지 업무 스킬과 5가지 슬래시 커맨드를 통해 반복적인 문서 작업, 보고서 작성, 번역, 데이터 분석 등을 자동화합니다.

한국어 비즈니스 문서의 형식·어투·산업 표준을 내장하고 있어, 외산 AI 도구가 놓치는 한국형 업무 맥락을 정확하게 처리합니다.

---

## 핵심 미션

반복 업무에 소모되는 시간을 줄이고, 직장인이 본질적인 업무에 집중할 수 있도록 합니다. 영어 중심으로 설계된 범용 AI 도구의 한계를 극복하여 한국 비즈니스 환경에 최적화된 AI 워크플로우를 제공합니다.

---

## 대상 사용자

### 직장인 (회사원)

- **상황**: 매주 반복되는 이메일 작성, 보고서 초안, 회의록 정리에 많은 시간 소요
- **니즈**: 빠른 문서 초안 생성, 격식 수준에 맞는 어투 선택, Excel 함수 자동화
- **활용 스킬**: 비즈니스 이메일, 회의록, 한국형 비즈니스 문서, 엑셀 업무 자동화, 주간/일일 보고서

### 취준생 (구직자)

- **상황**: 이력서·자기소개서 작성 경험 부족, 포트폴리오 구성 방법 불확실
- **니즈**: 직무별 맞춤 이력서, 자기소개서 검토 및 피드백, 링크드인 프로필 최적화
- **활용 스킬**: 이력서·포트폴리오, `/resume-check` 커맨드

### 1인사업자 (프리랜서·창업자)

- **상황**: 혼자서 제안서, 견적서, 계약 문서를 전부 처리해야 하는 부담
- **니즈**: 전문적인 제안서 템플릿, 견적서 자동화, PPT 발표자료 제작
- **활용 스킬**: 제안서/견적서, PPT 디자인 시스템, 프로젝트 관리, SVG 도식화

## 핵심 기능

### 12가지 업무 스킬

| 스킬 | 주요 기능 | 트리거 키워드 |
|------|-----------|---------------|
| **비즈니스 이메일** | 4단계 격식 수준 (격식/반격식/친근/공지), 상황별 템플릿 | 이메일, 메일, email |
| **회의록** | 자동 회의록 생성, 액션아이템 추출, 담당자 매핑 | 회의록, 회의, 미팅 |
| **한국형 비즈니스 문서** | 기획안, 보고서, 품의서, 결재 문서 형식 | 보고서, 기획안, 품의서 |
| **제안서/견적서** | 프로젝트 제안서, 사업 제안서, 견적서 자동 생성 | 제안서, 견적서, proposal |
| **엑셀 업무 자동화** | Excel 365 동적 배열 함수, 2026년 4대보험 요율, KPI 대시보드 | 엑셀, Excel, 스프레드시트 |
| **프롬프트 엔지니어링** | AI 활용 5원칙 + 26가지 기법 체계 | 프롬프트, prompt, AI 활용 |
| **프로젝트 관리** | WBS, 간트차트, RACI 매트릭스, 이슈 트래커 | 프로젝트, WBS, 일정 관리 |
| **이력서·포트폴리오** | 직무별 이력서, 자기소개서, 포트폴리오, 링크드인 최적화 | 이력서, 자기소개서, 포트폴리오 |
| **데이터 보고서** | 데이터 분석 서술, 시각화 차트, 인사이트 도출 | 데이터, 분석, 차트, 대시보드 |
| **한영 번역** | 7개 업종 전문용어 (IT, HR, 제조, 마케팅, 물류, 영업, 금융) | 번역, translate, 영작 |
| **PPT 디자인 시스템** | PptxGenJS 네이티브 요소 매핑, 4가지 컬러 테마, 한국형 레이아웃 | PPT 디자인, 슬라이드, 발표자료 |
| **SVG 도식화** | Mermaid v11 12종 다이어그램, 커스텀 SVG 비즈니스 도형 | 도식화, 다이어그램, SVG |

### 5가지 슬래시 커맨드

| 커맨드 | 설명 |
|--------|------|
| `/daily-report` | 오늘 업무 내용을 입력하면 일일업무보고서 즉시 생성 |
| `/weekly-report` | 주요 업무 키워드만으로 주간업무보고서 자동 작성 |
| `/quick-email` | 상황 설명만으로 격식에 맞는 이메일 즉시 생성 |
| `/meeting-note` | 회의 내용 요약 입력 시 구조화된 회의록 생성 |
| `/resume-check` | 이력서·자기소개서 업로드 후 항목별 피드백 제공 |

## 가치 제안

### 해결하는 문제

1. **언어·문화 장벽**: 영어 중심 AI 도구가 한국 비즈니스 어투, 존댓말 수준, 결재 문서 형식을 제대로 처리하지 못하는 문제를 해결합니다.

2. **반복 업무 부담**: 매주 반복되는 보고서, 이메일, 회의록 작성 시간을 대폭 단축합니다. 초안 작성 시간 기준 80% 이상 절감을 목표로 합니다.

3. **전문성 접근성**: 제안서 작성, PPT 디자인, 프로젝트 관리 방법론 등 전문 지식이 없어도 수준 높은 산출물을 만들 수 있습니다.

4. **최신 기준 반영**: 2026년 4대보험 요율, Excel 365 동적 배열 함수 등 최신 한국 업무 기준을 내장합니다.

### 차별화 요소

- **한국 비즈니스 특화**: 일반 AI 도구와 달리 한국 결재 문화, 회사 위계질서, 업종별 용어를 정확하게 이해합니다.
- **크로스 플랫폼**: Claude Code와 Cline 양쪽에서 동일한 기능을 사용할 수 있습니다.
- **즉시 사용 가능**: 복잡한 설정 없이 플러그인 설치만으로 바로 활용 가능합니다.
- **업종별 맞춤**: IT, 제조, 마케팅, 금융 등 7개 업종별 전문 용어 데이터베이스를 내장합니다.

## 주요 활용 사례

### 사례 1: 거래처 이메일 작성

```
입력: "거래처에 납품일정 변경 요청 이메일 써줘"
처리: 비즈니스 이메일 스킬 자동 활성화
결과: 격식 수준 선택 → 상황 설명 입력 → 완성된 이메일 즉시 생성
```

### 사례 2: 주간 보고서 생성

```
입력: /weekly-report 고객사 미팅 3건 완료, 제안서 초안 작성, 다음 주 최종 PT 예정
처리: 슬래시 커맨드 실행
결과: 주간 업무 현황, 성과, 다음 주 계획이 포함된 보고서 자동 생성
```

### 사례 3: 회의록 정리

```
입력: /meeting-note 참석: 김과장, 이대리. 3월 프로젝트 착수 합의, 예산 2억 확정.
처리: 회의록 스킬 활성화
결과: 안건, 논의 내용, 결정 사항, 액션아이템, 담당자 배정이 포함된 공식 회의록 생성
```

### 사례 4: Excel 급여 계산

```
입력: "2026년 기준 직원 급여에서 4대보험 공제액 계산하는 Excel 수식 만들어줘"
처리: 엑셀 업무 자동화 스킬 활성화
결과: 2026년 건강보험·국민연금·고용보험·산재보험 요율이 반영된 수식 및 함수 제공
```

### 사례 5: 이력서 검토

```
입력: /resume-check + 이력서 파일 첨부
처리: 이력서 검토 커맨드 실행
결과: 직무 적합성, 키워드 최적화, 자기소개서 개선 포인트 항목별 피드백
```

## 배포 방법

### Claude Code (플러그인 마켓플레이스)

1. Claude 데스크톱 앱 하단 **+** 버튼 클릭
2. **플러그인 추가** 선택
3. GitHub 마켓플레이스에서 `modu-ai/smart-cowork-life` 입력
4. **동기화** 후 **설치** 클릭
5. 12종 스킬 자동 활성화, 5종 슬래시 커맨드 즉시 사용 가능

### Cline (Rules 방식)

1. `.clinerules/` 디렉토리를 프로젝트 루트에 복사
2. Cline이 자동으로 Knowledge Rules(10종)를 상시 로드
3. 명시적 워크플로우(13종)는 사용자 요청 시 실행
